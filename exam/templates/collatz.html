Take n={{ challenge }}

Repeatedly apply the following transformation until you arrive at n==1

  • If n is even, divide by 2
  • If n is odd, multiply n by 3 and then add 1

What is the maximum number you've reached in the process ?

For example, starting at n==5, the sequence is 5, 16, 8, 4, 2, 1, and the answer is therefore 16.

P.S. If you ever find a number for which you don't end up at 1, please alert the nearest mathematician, you have just solved an open problem in mathematics.
